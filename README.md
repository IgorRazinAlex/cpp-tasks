# Как сдавать задачи:

1. Склонировать данный репозиторий
2. Переключиться на ветку с задачей
3. Разместить решение в `.h` файле, первой строчкой написать комментарий с id посылки в Яндекс.Контесте
4. Локально проверить, что решение проходит clang-tidy и clang-format:
```{bash}
# На примере deque
clang-tidy deque.h
diff deque.h <(clang-format deque.h)
```
5. git commit и push
6. Завести pull request на main ветку, в assignee и reviewer поставить проверяющего ассистента
7. Убедиться, что CI отработал без ошибок
8. Пингануть ассистента в телеграме со ссылкой на PR
9. После устранения замечаний повторить шаги 4-5, 7-8
